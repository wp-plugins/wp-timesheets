A simple timesheet app within WordPress Dashboard to manage time of your WordPress users.

== Description ==

A simple timesheet application within WordPress Dashboard to manage time of your blog users. Features include:

1. WordPress users can create and manage their own timesheets.
1. Admins can can create and manage timesheets of all users.
1. A range of customization options for Admins.
1. Customizable input fields to track extra hours.
1. Various reporting options including csv download.

== Installation ==

1. Upload folder `wp-timesheets` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
